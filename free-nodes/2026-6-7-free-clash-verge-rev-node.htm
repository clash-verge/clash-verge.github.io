订阅功能的重要性

使用Quantumult的订阅功能，最显著的优势便是它能够自动更新代理规则。对于大多数用户来说，手动更新代理配置既繁琐又耗时，而订阅功能解决了这一问题。通过订阅链接，Quantumult会定期自动更新代理规则，确保用户始终享有最新的代理服务。这样，无论是由于服务商更换节点，还是由于网络环境变化，用户都能迅速获得最新的网络资源配置。

订阅功能的另一个重要优势在于其极高的便捷性。用户只需在Quantumult的设置中输入一个有效的订阅链接，便可轻松导入多个代理规则，而不必逐一手动配置。这使得即便是技术水平一般的用户，也能够在短时间内完成科学上网的配置，快速开始畅享全球互联网。

如何使用Quantumult订阅功能

要充分利用Quantumult的订阅功能，用户首先需要了解如何添加、管理和更新订阅。以下是详细的操作步骤：

第一步：打开Quantumult设置

首先，用户需要打开Quantumult应用程序，并进入应用的“设置”选项。在设置界面，您将看到一系列配置选项，包括代理设置、订阅管理、日志记录等。

第二步：找到订阅管理

在设置界面中，找到“订阅”选项，并点击进入。在“订阅”页面中，用户可以查看当前已添加的订阅列表，并进行订阅链接的管理。

第三步：添加订阅链接

点击“添加订阅”按钮后，Quantumult会弹出一个对话框，用户需要在此输入指定的订阅链接。订阅链接通常由第三方提供，用户可以选择公共订阅链接，也可以购买付费订阅服务。一旦输入订阅链接并点击“确定”按钮，Quantumult将自动添加订阅并开始同步配置文件。

第四步：更新订阅

一旦成功添加订阅链接，用户可以在订阅界面中看到已添加的订阅列表。点击“更新”按钮，Quantumult将连接至订阅链接，获取并下载最新的代理规则。通过这种方式，用户可以确保始终使用最新的代理配置，避免因规则过时而导致连接不稳定或速度变慢的问题。

公共订阅是指免费提供给广大用户使用的订阅服务。这些订阅通常会提供基础的代理功能，适合一般用户进行日常使用。公共订阅的优点是免费且易于获取，但其服务器数量和速度可能相对较慢，连接稳定性也可能不如付费订阅。

付费订阅

付费订阅是为高需求用户提供的专业服务。通过购买付费订阅，用户能够享受到更快的连接速度、更高的稳定性以及更完善的隐私保护功能。付费订阅通常提供多个可选节点，用户可以根据自身的需求选择合适的服务器位置。

定制订阅

FAQ

Shadowrocket 节点测速结合策略组的作用？
Shadowrocket 定期测速节点延迟和丢包率，并结合策略组自动选择最佳节点。保证网页、视频和游戏流量走低延迟节点，提高访问速度和连接稳定性。
Nekoray 支持节点订阅的自动更新吗？
支持。用户可通过订阅 URL 批量更新节点，程序会自动下载并替换旧节点。定期自动更新可保证节点列表始终有效，减少手动维护的工作量，提高使用效率。
Sing-box 的配置文件与 Clash 有何区别？
Sing-box 的配置使用 JSON 格式，而 Clash 采用 YAML 格式。前者更适合程序自动解析，后者更便于人工编辑。Sing-box 在性能和扩展性上有明显优势，支持更细粒度的流量控制。
Matsuri 支持哪些传输方式？
Matsuri 支持 TCP、gRPC、QUIC、WebSocket 等多种传输协议。用户可在配置文件中自由切换，根据服务器支持情况选择最优连接方式，以获得更低延迟与更强稳定性。
SSR 是否支持 UDP 协议转发？
原版 SSR 并不完全支持 UDP 转发，但部分魔改版本（如 SSRR）支持该功能。可在配置中开启 udp-over-tcp 模式或使用 Socks5 UDP relay 方案实现游戏等实时通信的代理需求。
Clash 的延迟检测如何帮助节点选择？
Clash 会定期测速节点延迟，并根据策略组自动选择延迟最低节点处理流量。可避免访问高延迟或失效节点，提高网页、视频和游戏访问速度及稳定性。
Sing-box 的 JSON 配置文件常见语法错误如何快速定位？
通过 JSON 校验工具或编辑器（例如 VSCode）打开配置文件，查看语法高亮与错误提示；也可运行 sing-box 并观察控制台错误信息，错误通常会指出第几行或哪个字段不合法以便修正。
Netch 是否可以代理局域网设备？
可以。Netch 支持启用局域网共享功能，使同一网络下的其他设备通过主机的代理访问外网。需确保防火墙允许入站连接，且代理端口在局域网中开放。
Hysteria2 多路复用在短连接中优势是什么？
多路复用允许多个短连接共享同一通道，减少握手延迟和资源消耗。适合高频短连接场景，如游戏和即时通讯，提升吞吐量和连接稳定性，优化网络体验。
Potatso Lite 支持手动和二维码导入节点吗？
支持。用户可以手动添加节点信息，或者通过扫描节点二维码导入节点。虽然不支持自动订阅更新，但足够满足轻量用户对节点管理和分流的需求。
